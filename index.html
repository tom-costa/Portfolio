Destination Checker

A search tool I created to help research and check potential destinations I want to travel to. I leveraged the below API's to display results from a search:

  • Open Weather Map - To Fetch the Weather.
  • YouTube - To show travel guide videos of the destination.
  • Open Cage Data - Geolocation (to convert your coordinates (lat & lon)) to get the user's city.
  • Unsplash - To dynamically pull in an image of the city searched, for inspiration.
  • Wikipedia - To bring up articles related to your destination.

GRATITUDE Space

I created this CRUD web app to help me reflect daily on the things I am grateful for. I wanted to create a "zen" atmosphere, so I added features to let the user change the animated background and song to fit their mood. This project had heavy emphasis on user experience and keeping the code modular (separate JS file for each functionality + use of CSS variables.)
Features:

  • Local Storage - Gratitude items can be created, read, updated and deleted. All saved to Local Storage.
  • Motivational Quotes - Each gratitude item triggers a random motivational quote.
  • Animated Background - User has a range of backgrounds to choose from (from Pexels).
  • Music Player - I curated a list of royalty-free songs the user can choose from.
